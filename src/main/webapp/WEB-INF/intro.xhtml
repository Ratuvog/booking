Добро пожаловать!

Данная система обработки заказов реализована при помощи Java Spring Web Flow

В приложении доступны следующие функции

  • Поиск отелей по названию, городу, стране
  • Постраничный вывод результатов поиска
  • Сохранение состояния выполнения процесса бронирования
  • Просмотр и отмена предыдущих заказов
  • Просмотр детальной информации об отелях
  • Оформление брони в выбранном отеле
